# Contributing to llm-queries

Thanks for helping improve this repository. The goal is to keep prompts, guides, and references clear, accurate, and easy to navigate.

## Ways to contribute

- Fix typos or broken links
- Add new prompts, guides, or course reviews
- Improve structure and navigation
- Add citations or clarify technical details

## Quick start

1. Fork the repo and create a branch: `git checkout -b your-change`
2. Make changes with a focused scope
3. Verify any links you touched
4. Open a PR with a short summary and rationale

## Repository layout

- `prompts/` prompt templates by category
- `local_setup_guides/` local LLM setup and optimization guides
- `resources/` external references and reading
- `course_reviews/` course notes and summaries
- `slides/` slide decks and lecture-style notes

## Content standards

- Add a short intro at the top explaining purpose and intended audience
- Use consistent headings (`#` for title, `##` for sections)
- Include citations for factual claims and benchmarks
- Prefer ASCII file names with kebab-case
- Keep prompts concise and testable

## PR checklist

- New or updated files have a clear introduction
- Links are valid
- Examples are runnable or clearly marked as pseudo
- Claims include sources

## Questions

Open an issue if you want feedback on a proposal before doing the work.
