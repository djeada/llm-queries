### 1. Current information / news search

> “Search the web and summarize the latest major developments in AI regulation in the EU. Use at least 3 reliable sources and include links.”

Tests: web search, source quality, recency, citations.

### 2. Product comparison

> “Find 5 mechanical keyboards under $100, compare specs, prices, reviews, and give purchase links.”

Tests: shopping search, filtering, ranking, links, price awareness.

### 3. Travel planning

> “Plan a 3-day weekend trip to Prague from Berlin next month. Find train options, hotels under €120/night, and 5 things to do.”

Tests: web search, date handling, maps/travel logic, constraints.

### 4. Weather-aware planning

> “Check the weather in Munich this weekend and suggest what clothes I should pack.”

Tests: weather tool, location/time awareness, practical reasoning.

### 5. Calendar task

> “Check my calendar for next Tuesday and find a free 1-hour slot between 10 AM and 5 PM.”

Tests: calendar access, availability reasoning, time zones.

### 6. Email search

> “Search my email for the most recent invoice from Amazon and summarize the amount, date, and order details.”

Tests: Gmail search, reading messages, extracting structured info.

### 7. File/document analysis

> “Find the PDF I uploaded about machine learning and summarize the key points with citations from the document.”

Tests: file search, document reading, citations, summarization.

### 8. Spreadsheet generation

> “Create a spreadsheet comparing 10 cloud hosting providers by price, free tier, storage, bandwidth, and best use case.”

Tests: spreadsheet creation, structured data, formatting.

### 9. Slides generation

> “Create a 6-slide presentation explaining Docker to beginners, with diagrams and speaker notes.”

Tests: artifact creation, structure, formatting, visual planning.

### 10. Code execution / data analysis

> “Use Python to analyze this CSV, find trends, create a chart, and explain the result.”

Tests: Python tool use, data cleaning, charts, interpretation.

### 11. Multi-tool task

> “Find the cheapest flight from Berlin to Barcelona next month, check the weather there, and create a packing checklist.”

Tests: combining web + weather + reasoning.

### 12. Verification task

> “Tell me who the current CEO of OpenAI is, verify it from official or reliable sources, and include links.”

Tests: current facts, browsing, not relying on stale memory.

### 13. Map/local search

> “Find 5 highly rated coffee shops near Alexanderplatz open after 8 PM today.”

Tests: local search, current hours, location reasoning.

### 14. Automation/reminder

> “Remind me every Monday at 9 AM to submit my weekly report.”

Tests: scheduled task creation.

### 15. Image generation/editing

> “Generate an image of a futuristic laptop workspace in a cyberpunk style.”

Tests: image generation tool call.

A very good benchmark set would include one task each for **web search, shopping, weather, calendar, email, files, spreadsheet, slides, Python, image generation, and automation**.
