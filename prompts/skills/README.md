# Skills Prompt Library

Prompt templates for assessing skills, planning learning paths, practicing deliberately, and turning practice into evidence for resumes, portfolios, or performance reviews.

## Best for

- Identifying skill gaps for a target role or project
- Building a realistic learning plan
- Creating practice exercises with feedback rubrics
- Translating completed work into portfolio, resume, or review-ready evidence

## Prompts

| Prompt | What It Does | Best For |
|--------|--------------|----------|
| [skill_gap_analysis.md](skill_gap_analysis.md) | Compares current ability against a role, project, or goal | Career planning, upskilling |
| [learning_plan.md](learning_plan.md) | Creates a structured learning roadmap | Self-study, team training |
| [deliberate_practice.md](deliberate_practice.md) | Generates drills, exercises, and feedback rubrics | Skill improvement |
| [portfolio_evidence.md](portfolio_evidence.md) | Turns skill work into proof of ability | Resumes, portfolios, reviews |

## Recommended Workflow

1. Start with `skill_gap_analysis.md` to clarify what matters.
2. Use `learning_plan.md` to create a realistic roadmap.
3. Use `deliberate_practice.md` for exercises and feedback.
4. Use `portfolio_evidence.md` to document outcomes.

## Quality Checklist

- The prompt defines the target role, project, or outcome.
- The model separates current ability from desired ability.
- Recommendations are realistic for the available time.
- Practice tasks include observable success criteria.
- Evidence claims are grounded in completed work, not aspirations.
